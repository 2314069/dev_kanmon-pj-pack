# ドキュメントテンプレート

> 感門準備で使用するExcel・PowerPointテンプレートの格納場所です。

## テンプレート一覧

| ファイル名 | 用途 | 使用タイミング |
|-----------|------|--------------|
| 座席表ひな型.pptx | 当日の座席配置図 | 参加者FIX後 |
| 参加者名簿.xlsx | 参加者リスト管理 | 準備期間中随時更新 |
| 次第.xlsx | 感門当日の次第表 | 次第作り時 |
| ロール表.xlsx | 感門団のロール割り振り | 感門団ロール決定時 |

## 使い方

1. テンプレートをコピーして使用
2. 期ごとのプロジェクトフォルダ（05_プロジェクト履歴）に保存
3. BOXにアップロードして共同編集

## 注意事項

- テンプレート本体は編集しないでください
- 実際のファイルはBOXで管理します
- ローカル編集時は [ルール.md](../../01_ルル/ルール.md) を参照
