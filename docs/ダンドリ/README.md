## 準備タイムライン

感門之盟に向けた準備の全体スケジュールです。

```
【3ヶ月前】────────────────────────────────
  ├─ 感門準備統括のアサイン
  ├─ 感門ボード（ラウンジ）立ち上げ
  └─ サブチーム編成開始
      └─ 感門団／エディスト／冊子／アフ感

【2ヶ月前】────────────────────────────────
  ├─ 感門テーマ決定
  ├─ 冊子編集部キックオフ
  ├─ MCラウンジ立ち上げ
  └─ 各サブチームのロール決定

【1ヶ月前】────────────────────────────────
  ├─ 感門団ロール表作成
  ├─ エディスト事前記事の配信開始
  ├─ 冊子原稿収集・ディレクション
  └─ MCキックオフ・原稿準備

【2週間前】────────────────────────────────
  ├─ 次第FIX
  ├─ 座席表・受付名簿作成
  ├─ 参加者人数FIX
  └─ 冊子入稿

【1週間前】────────────────────────────────
  ├─ 本楼リハーサル
  ├─ 名札・配布物準備
  ├─ 物品の在処確認
  └─ 最終確認MTG

【前日】──────────────────────────────────
  ├─ 会場準備（掃除・設営）
  └─ 最終チェック

【当日】──────────────────────────────────
  ├─ 朝会（全員集合・最終確認）
  ├─ 感門之盟本番
  └─ アフ感

【翌週】──────────────────────────────────
  └─ 振り返りMTG
```

{% hint style="info" %}
**ポイント**

各チームの詳細スケジュールは、それぞれのダンドリページを参照してください。
{% endhint %}

## メインタスク大項目

- ロールアサイン
- 感門ボード立ち上げ
- サブチーム立ち上げ
    - 感門団
    - 感門エディスト
    - MCラウンジ
    - アフ感
    - 次第作り
    - テーマ決め
    - 冊子編集
- 感門当日の人数決定
    - 学衆参加者
    - 当期指導陣参加者
    - 感門団参加者
    - エディストライター
    - 学林局関係者
    - その他
    - 感門当日の1週間ほど前にFIXする予定
- 事前準備
- 事前リハ
- 当日
- 振り返り

## 全体ダンドリ

1. **感門タイトル決定**
    - 与件の整理

2. **感門団ラウンジ作成**

3. **メンバー登録**
    - 関係講座の師範、番匠、花目付、学匠、学林局
    - JUST記者との連携者

4. **ラウンジ開設の投稿**
    - ラウンジの位置付け
    - ラウンジメンバー
    - 目的の共有
    - ルール（Box案内、ドキュメント更新など）の共有

5. **サブチーム作成**（統括＋講座PL＋チームメンバー）
    - 感門団チーム
    - エディストチーム
    - 冊子チーム
    - アフ感チーム

6. **ロール決定**
    - 講座ごとの感門団PL決定
    - サブチームごとのメンバー募集
    - 総合司会

7. **サブチームごとにダンドリ進捗**

8. **事前準備**

9. **事前リハ**

10. **当日**

11. **振り返り**
